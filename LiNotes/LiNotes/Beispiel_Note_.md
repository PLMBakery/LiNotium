# Beispiel Note 

Beispiel-Notiz für Markdown
Einführung
Markdown ist eine einfache Auszeichnungssprache, die in reinen Textformaten geschrieben wird. Es wird häufig verwendet, um formatierten Text in einer einfachen und leicht lesbaren Form zu erstellen.

Grundlegende Formatierungen
Fett und Kursiv
Fett: **Text** oder __Text__
Kursiv: *Text* oder _Text_
Fett und Kursiv: ***Text*** oder ___Text___
Überschriften
Markdown unterstützt sechs Ebenen von Überschriften, die durch eine unterschiedliche Anzahl von Hash-Zeichen (#) angezeigt werden:

# H1
## H2
### H3
#### H4
##### H5
###### H6
Listen
Ungeordnete Liste
Punkt 1
Punkt 2
Unterpunkt 2.1
Unterpunkt 2.2
Verwendet wird:

markdown
Code kopieren
- Punkt 1
- Punkt 2
  - Unterpunkt 2.1
  - Unterpunkt 2.2
Geordnete Liste
Punkt 1
Punkt 2
Unterpunkt 2.1
Unterpunkt 2.2
Verwendet wird:

markdown
Code kopieren
1. Punkt 1
2. Punkt 2
   1. Unterpunkt 2.1
   2. Unterpunkt 2.2
Links und Bilder
Linktext
Verwendet wird:

markdown
Code kopieren
[Linktext](https://example.com)
![Alt-Text](https://example.com/bild.jpg)
Zitate
Dies ist ein Zitat. Zitate werden durch das Zeichen > erstellt.

Verwendet wird:

markdown
Code kopieren
> Dies ist ein Zitat. Zitate werden durch das Zeichen `>` erstellt.
Code
Inline-Code
Verwende backticks für Inline-Code: printf("Hello, World!");

Verwendet wird:

markdown
Code kopieren
Verwende `backticks` für Inline-Code: `printf("Hello, World!");`
Codeblöcke
python
Code kopieren
def hello_world():
    print("Hello, World!")
Verwendet wird:

scss
Code kopieren
```python
def hello_world():
    print("Hello, World!")
```
